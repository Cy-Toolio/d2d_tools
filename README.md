# d2d_tools

A collection of day-to-day tools for data work — quick scripts for analysis, manipulation, and exploration that don't belong in a larger codebase.

## Tools

| Tool | Description |
|------|-------------|
| [csv_tui](./csv_tui/) | Terminal-based CSV analyzer with filtering, compute columns, and distinct value views |

## Adding a tool

Drop a new folder at the repo root with the tool file(s) and a `README.md` covering usage.
